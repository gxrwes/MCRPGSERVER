Detailed Description

A CommandNode that is triggered by an argument, like a number.

The type of the argument is set via an ArgumentType, which takes care of parsing the stringified argument passed by users.


A short example:
A subcommand for a delete command, taking the number of messages to delete. The ArgumentType would be an IntegerArgumentType then and the built command would parse things like {@literal "20"} or {@literal "-230"}.

Parameters
<S>the type of the command source
<T>the ArgumentType the built command will use
